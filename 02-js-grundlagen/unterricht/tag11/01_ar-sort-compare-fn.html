Array - ar.sort(compareFn)

ar.sort(compareFn) - Die sort() Methode sortiert die Elemente eines Arrays in-place und gibt das Array zurück. Standardmäßig werden alle Elemente in Strings umgewandelt und dann anhand ihrer UTF-16 Codepoints miteinander verglichen.

Das angesprochene Array wird durch in-place-mutation ebenfalls sortiert.

Rückgabewerte der Vergleichfunktion:

  • Rückgabewert 0: Positionen bleiben stehen
  • Rückgabewert < 0: Positionen bleiben stehen
  • Rückgabewert > 0: Positionen vertauscht
