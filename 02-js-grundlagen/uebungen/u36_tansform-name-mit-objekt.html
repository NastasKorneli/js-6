Übung 36: Vorname vor Nachname oder Nachname vor Vorname?

Wenn Sie der Name der Übung beim Lesen genauso verwirrt wie uns beim Schreiben, dann liegt das daran, dass die Reihenfolge von Vor- und Nachnamen nicht immer eindeutig ist. Machmal heißt es Max Mustermann und manchmal auch Mustermann, Max.
Damit Benutzer der Funktion transformName (aus Übung 36) nicht auch so verwirrt sind, stellen Sie sie besser auf named parameters um.
