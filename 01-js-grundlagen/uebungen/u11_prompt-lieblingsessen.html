Übung 11: Mit prompt das Lieblingsessen erfragen

  1. Fragen Sie den Benutzer nach seinem Lieblingsessen und seinem Lieblingsgetränk und speichern Sie beides in passend benannte Variablen.
  2. Verwenden Sie beide Variablen in einem Satz, den Sie über die Konsole ausgeben. Verketten Sie dabei die Variablen mithilfe des Verkettungsoperators mit den restlichen Satzteilen .
  3. Führen Sie die letzte Übung noch einmal durch, aber verwenden Sie dieses Mal Template String Substitutions.
  4. Wenn Sie ganz fleißig sind, können Sie als dritte Variante noch austesten, wie sich beide Varianten in der gleichen Ausgabe verwenden lassen.
