Literale

Grundsätzlich gilt, dass jeder Wert — egal ob String oder Zahl — der direkt wörtlich (engl.: literally) im Code steht, als Literal bezeichnet wird. Literale haben immer einen festen Wert.

Beispiele für Literale

  • 'Haus'
  • 'grün'
  • 5.47
  • 1998

  • true
